# AGENTS.md

The contribution guide is in [`CONTRIBUTING.md`](CONTRIBUTING.md). Read and follow it. 
This file adds only the rules specific to working as an AI agent in a session.

## Behavioural

- If the user asks a question, **answer it — do not write to files**. Discussion
  is not a change request.
- Do **not** proactively create or edit files unless explicitly requested.
- Discuss freely and in depth; ask clarifying questions when exploring options.
- Do **not** agree for the sake of agreement. Push back when there are grounds.
- **Decisions are the user's domain.** Bring options and trade-offs; do not
  decide unilaterally.
- **Wait for review before creating commits.** Do not commit unprompted.
- Do not install packages or change the environment without confirming.
- Do not create extra branches unless asked.
